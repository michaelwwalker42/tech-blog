# Tech Blog
  ![license](https://img.shields.io/badge/license-MIT-orange.svg)

  ## Description
  A CMS-style blog site , where developers can publish their blog posts and comment on other developersâ€™ posts as well. The app uses Handlebars.js as the templating language, Sequelize as the ORM, and the express-session npm package for authentication. The site is hosted by [Heroku](https://www.heroku.com), and uses [JawsDb](https://elements.heroku.com/addons/jawsdb) MySQL database.
